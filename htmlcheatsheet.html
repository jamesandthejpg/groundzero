HTML Tags Cheat Sheet

A simple beginner-friendly guide to common, lesser-used, and niche HTML tags, separated clearly so they are easier to learn.

  • Common Tags
  • Lesser-Used Tags
  • Rare / Niche Tags
  • Quick Tips

Common HTML Tags

These are the tags you will use most often in basic websites.

Page Structure

<!DOCTYPE html> <html> <head> <body>

These tags build the basic skeleton of every HTML page.

<!DOCTYPE html> <html> <head></head> <body></body> </html>

Text Content

<h1> to <h6> <p> <br> <hr>

Used for headings, paragraphs, line breaks, and section dividers.

<h1>Main Title</h1> <p>This is a paragraph.</p>

Links and Images

<a> <img>

These let you link to other pages and display images.

<a href="https://example.com">Visit</a> <img src="photo.jpg" alt="Description">

Lists

<ul> <ol> <li>

Used for unordered and ordered lists.

<ul> <li>Item</li> </ul>

Layout and Sections

<div> <span> <header> <main> <section> <footer>

Helpful for grouping content and making your page easier to structure.

<section> <h2>About</h2> <p>Content here</p> </section>

Forms

<form> <input> <button> <label>

Used when collecting information from users.

<form> <label>Name</label> <input type="text"> <button>Submit</button> </form>

Lesser-Used but Useful Tags

These are great once you want your site to feel more polished and semantic.

Extra Semantic Tags

<article> <aside> <nav>

These describe the role of your content better than plain divs.

Figures and Captions

<figure> <figcaption>

Useful for images, diagrams, and content that needs a caption.

<figure> <img src="code.png" alt="Code example"> <figcaption>A code example</figcaption> </figure>

Expandable Content

<details> <summary>

Creates built-in dropdown style sections without JavaScript.

<details> <summary>Open me</summary> Hidden content here </details>

Text Meaning

<mark> <small> <abbr> <code> <pre>

These help give extra meaning or formatting to text.

Tables

<table> <thead> <tbody> <tr> <th> <td>

Useful for organised data, not for page layout.

Advanced Input Types

type="email" type="date" type="range" type="color"

These make forms much more interactive and user-friendly.

Rare / Niche Tags

These are worth knowing about, but you probably will not use them often in beginner projects.

Popups and Indicators

<dialog> <progress> <meter>

Useful for modals, loading bars, or measurements.

Special Text Tags

<time> <sup> <sub> <bdi>

These help with dates, formatting, and text direction handling.

Hidden or Script-Related

<noscript> <template> <canvas>

Mostly used with JavaScript for advanced interactions or fallback content.

Deprecated Tags to Avoid

<center> <font>

These are outdated. Use CSS instead for alignment and styling.

Quick Tips

  • Use semantic tags like <section> and <article> when possible.
  • Always add alt text to images.
  • Use <div> for layout, but not for everything.
  • Use CSS for styling instead of old HTML tags like <font>.
  • Start with the common tags first, then add the lesser-used ones as you improve.
